# CC-DSW

## Setups
The requiring environment is as bellow:  

- Linux 
- Python 3.8.18
- PyTorch 1.8.1 
- Torchvision 0.9.1


## Running CC-DSW on benchmark datasets (PCam and ISIC 2020).
Here is an example:
```bash
python train.py --dataset PCam --model res32
```
